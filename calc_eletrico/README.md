# Calculadora de Dimensionamento Elétrico ⚡

Software em Python para dimensionamento de cabos elétricos,
sugestão de disjuntores, cálculo de potência e geração
automática de relatório técnico em PDF.

## Funcionalidades
- Dimensionamento de cabos por corrente
- Sugestão de disjuntores padrão
- Curvas B, C e D
- Interface gráfica (Tkinter)
- Geração automática de PDF

## Tecnologias
- Python
- Tkinter
- ReportLab

## Observação
Projeto com fins educacionais.
